Milling Company Production Planning Model

Chicken Meat Quality Control App

This is a PowerPlatform solution for a fictional chicken company's production site quality control operations. It focusses on quality parameter data capturing.

Power-platform

About this project

A large part of quality control involves records management. It is said in the quality control circles that if there is no record, then there is no evidence of quality control. However, manual paperwork in data capturing can be cumbersome as physical records are difficult to manage and can easily be lost.

This app fixes that problem. Data is captured electronically using a phone/tablet and is stored on a sharepoint list storage. It is easily accessible for future tracing and data analysis.

Tools used

PowerApps SharePoint Lists

Intercity Bus Logistics Dashboard

This simple one page dashboard models the Zambian intercity bus logistics business. It captures operational metrics for easy monitoring.

PowerBI

About this project

This dashboard gives insight into the main operational KPIs for the Zambian intercity bus logistics busniess. It provides insight on revenue, number trips, trip durations, and station fees by route and by bus. Additionally, maintenan costs per bus are included to ease the process of route cause analysis when needed.

The model is two dimentional, showing the magnitute of these KPIs over time. This is great for showing the company's growth, especially to important stakeholders and would be investors. It also flags danger in the case that the company may be performing below previously achieved levels.
