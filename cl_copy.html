Sales Operations Dashboard

End-to-end Power BI dashboard for regional sales performance tracking, with automated refresh via Power Automate.

Power Platform

About this project

Built for a FMCG client, this solution consolidates data from SharePoint Lists and SQL Server into a live Power BI report. Power Automate handles nightly data refresh and sends exception alerts to the regional sales team via Teams. The dashboard reduced time-to-insight from two days to under fifteen minutes.
